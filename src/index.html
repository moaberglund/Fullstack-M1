Om uppgiften

Detta är en webbplats skapad som första delmoment i kursen Fullstack utveckling, som är del av programmet Webbutveckling vid Mittuniversitetet. Dett moment går ut på att utforka olika CSS ramverk samt olika platformar man kan publicera till.

CSS ramverk

På sida 1 har jag använt mig av Milligram, på sida 2 har jag använt mig av Bootstrap. Jag provade även på att använda Tailwind på Publiceringssidan samt Foundation här på startsidan för att få lite mer förståelse för olika ramverk som finns tillgängliga.
