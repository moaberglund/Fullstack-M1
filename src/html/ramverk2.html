Bootstrap

Om ramverket

Denna sidan har använt sig av ramverket Bootstrap. Det var enkelt att sätta upp och komma igång med. När man redan har en del kunskap om styling i CSS kan man enkelt googla runt efter olika sätt att implementera tex padding och margin. Och Bootstraps egen hemsida har utförlig dokumentation om ramverket och bra sätt att använda det på inklusive exempel.

Nu är jag i och för sig mycket nybörjare på detta ramverk men föredrar absolut SCSS över detta. Känns ganska jobbigt att allting ligger i HTML kode, blir rörigt och svårare att ändra i efterhand. Jag gillar det dock för formulär och tabell vilket jag aldrig varit speciellt bra på att styla i CSS. Så den delen är absolut en vinst!

Installation

Parcel

npm install bootstrap@5.3.3

CDN

Lägg till Bootstrap stylesheet i din header tagg

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">

Lägg till Bootstrap JavaScript i slutet av din body tagg

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js" integrity="sha384-YvpcrYf0tY3lHB60NNkmXc5s9fDVZLESaAA55NDzOxhy9GkcIdslK1eN7N6jIeHz" crossorigin="anonymous"></script>
