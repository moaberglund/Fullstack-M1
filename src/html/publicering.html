Publicering

Vercel

För publicering använde jag Vercel.

När man publicerar ett Parcel projekt så behöver källan för publicering vara rooten av mappen "/." Det fungerade sömnlöst att ladda upp och tjänsten har automatiserad publicering, den lyssnar alltså på Github repot efter nya commits och uppdaterar applikationen om nya ändringar gjorts.

Vercel har både gratis och betal varianter beroende på vad du vill få ut från din webbplats du lägger upp via dem.

Förutom Parcel som jag har använt mig av i detta projekt så har Vercel även stöd för en hel del andra ramverk, några exempel är:

  • React
  • Vue
  • Angular
  • Next
  • Nuxt
  • Svelte
  • Vite
  • Astro

Azure

Provade även publicera med Azure men lyckades tyvärr inte få igenom publicering där, behöver gräva lite djupare i hur man publicerar bäst med den platformen. Stylingen hängde inte med när jag publicerade men har i alla fall kommit ett första steg på vägen med att sätta upp konto och utforska webbplatsen lite.

Tailwind

Denna undersida har skapats med hjälp av CSS ramverket Tailwind. Hade problem med npm installationen men rullade på fint när jag bytte till CDN istället. Tailwind är ett smidigt sätt att styla sin webbplats på, speciellt om man inte är så bekant med CSS styling sedan innan. Det har varit nice att få en känsla av hur det fungerar.
